## Компоненты

### `<App />`

Корневой компонент. Рендерит заголовок, `<NotificationPanel />` и `<HomeworkList />`.

---

### `<NotificationPanel />`

**Данные:** `GET /api/next-notification` → `{ scheduledAt: string | null }`

**Лонг-поллинг:** после получения ответа делает следующий запрос через 10 секунд (простой интервал, не настоящий long-poll). Если `scheduledAt === null` — показывает «Отправка не запланирована».

**Содержимое:**

1. **Строка с датой** — «Следующая отправка: чт 02.04 в 17:30»
2. **Статус-бар** — полоса заполнения от 0% до 100%:
   - 0% — прямо сейчас (сразу после предыдущей отправки, т.е. далеко от `scheduledAt`)
   - 100% — когда `scheduledAt` наступило
   - Логика: заполнение = `1 - (msLeft / totalMs)`, где `totalMs` — интервал от начала суток до `scheduledAt` (или фиксированные 24 ч, если хочется проще)
   - Визуально: цвет плавно меняется от зелёного → жёлтого → оранжевого → красного по мере заполнения
3. **Кнопка «Отправить сейчас»** — открывает `<ConfirmModal />`

---

### `<ConfirmModal />`

Появляется поверх страницы (backdrop с blur).

**Содержимое:**
- Текст: «Введите слово подтверждения»
- `<input>` для ввода кода
- Кнопка **Отправить**

**Логика:**
- При нажатии «Отправить» делает `POST /api/send-now` с телом `{ "code": "<введённое значение>" }`
- Если ответ `200` — закрыть модалку, показать тост «Отправлено!»
- Если ответ `403` — показать ошибку внутри модалки «Неверный код»
- Если `404` — «Нет домашнего задания»
- Пока идёт запрос — кнопка задизейблена и показывает лоадер

---

### `<HomeworkList />`

**Данные:** `GET /api/available-days` → массив:
```json
[
  {
    "date": "01.04.2026",
    "homework": [
      { "subject": "Математика", "task": "с. 72 (р.т.)" }
    ]
  }
]
```

Рендерит список дней. Каждый день — блок с датой-заголовком и списком предметов.

Запрос делается один раз при монтировании. Если массив пустой — «Домашнее задание не найдено».

## API

Базовый URL берётся из env-переменной `VITE_API_BASE_URL` (например `http://localhost:3000`).

| Метод | Путь                     | Описание                              |
|-------|--------------------------|---------------------------------------|
| GET   | `/api/next-notification` | Дата/время следующей запланированной отправки |
| POST  | `/api/send-now`          | Отправить сейчас. Body: `{ code: string }` |
| GET   | `/api/available-days`    | Список дней с домашним заданием       |
